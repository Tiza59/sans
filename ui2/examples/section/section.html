Sans UI Section Example

This example demonstrates how to use the Sans UI Section component in a plain HTML page.

Basic Section

Light Theme Section

This is a basic Sans UI Section component with light theme styling.

 
<sans-section className="content-section light-theme">
  <h3>Light Theme Section</h3>
  <p>This is a basic Sans UI Section component with light theme styling.</p>
</sans-section>

Section with Custom Attributes

Dark Theme Section

This section has a custom ID, class, role, and aria-label.

 
<sans-section 
  id="custom-section" 
  className="content-section dark-theme" 
  role="region" 
  ariaLabel="Dark theme section"
>
  <h3>Dark Theme Section</h3>
  <p>This section has a custom ID, class, role, and aria-label.</p>
</sans-section>

Nested Sections

Parent Section

This is a parent section that contains a nested section.

Nested Section

This is a nested section with accent styling.

 
<sans-section className="content-section light-theme">
  <h3>Parent Section</h3>
  <p>This is a parent section that contains a nested section.</p>
  
  <sans-section className="content-section accent-theme">
    <h3>Nested Section</h3>
    <p>This is a nested section with accent styling.</p>
  </sans-section>
</sans-section>

Event Handling

Interactive Section

Click me to see an event in action.

 
<sans-section 
  className="content-section accent-theme" 
  id="click-section" 
  style="cursor: pointer;"
>
  <h3>Interactive Section</h3>
  <p>Click me to see an event in action.</p>
</sans-section>

<script>
  const section = document.getElementById('click-section');
  section.addEventListener('click', () => {
    document.getElementById('output').textContent = 
      'Section clicked at: ' + new Date().toLocaleTimeString();
  });
</script>
