Sans UI Div Example

This example demonstrates how to use the Sans UI Div component in a plain HTML page.

Basic Div

This is a basic Sans UI Div component with custom styling 
<sans-div className="box primary-bg">
  This is a basic Sans UI Div component with custom styling
</sans-div>

Div with ID and Attributes

This div has a custom ID, class, role, and aria-label 
<sans-div 
  id="custom-div" 
  className="box secondary-bg" 
  role="region" 
  ariaLabel="Example region"
>
  This div has a custom ID, class, role, and aria-label
</sans-div>

Nested Divs

Item 1 Item 2 Item 3 
<sans-div className="flex-container">
  <sans-div className="flex-item">Item 1</sans-div>
  <sans-div className="flex-item">Item 2</sans-div>
  <sans-div className="flex-item">Item 3</sans-div>
</sans-div>

Event Handling

Click me to see an event in action 
<sans-div 
  className="box secondary-bg" 
  id="click-div" 
  style="cursor: pointer;"
>
  Click me to see an event in action
</sans-div>

<script>
  const div = document.getElementById('click-div');
  div.addEventListener('click', () => {
    document.getElementById('output').textContent = 
      'Div clicked at: ' + new Date().toLocaleTimeString();
  });
</script>
